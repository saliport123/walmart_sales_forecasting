
Walmart Sales Forecasting Project

Files Included:
- sales_forecasting.py : Python code for data preprocessing, EDA, model training.
- correlation_heatmap.png : Heatmap of numerical feature correlations.
- weekly_sales_trend.png : Sales trends of top 3 stores.
- README.txt : Project overview and instructions.

How to Run:
1. Place 'train.csv', 'features.csv', 'stores.csv' in the same folder.
2. Run `sales_forecasting.py` to train and evaluate the model.
